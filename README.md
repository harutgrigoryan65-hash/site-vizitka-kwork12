# LiveFrame Production

Статический одностраничный сайт для направления многокамерных съемок, технического продакшена и онлайн-трансляций.

## Структура

- `index.html` - HTML-разметка страницы.
- `styles.css` - стили и адаптивная сетка.
- `script.js` - обработка формы заявки и отправка данных.

## Запуск

Сайт не требует сборки. Для локального просмотра достаточно открыть `index.html` в браузере.

## Форма заявки

Форма находится в блоке `#contact` и обрабатывается файлом `script.js`.

Основные поля:

- имя;
- телефон или email;
- формат события;
- дата и город;
- описание задачи;
- источник заявки.

Для отправки данных во внешнюю систему используется переменная:

```js
const WEBHOOK_URL = "";
```

При заполненном `WEBHOOK_URL` форма отправляет POST-запрос в JSON-формате. Если endpoint не указан или недоступен, данные сохраняются локально в браузере, а пользователь получает статус отправки.

## Интеграция

Форму можно подключить к CRM или автоматизации через webhook. Поддерживаемый сценарий передачи данных: браузерная форма -> webhook -> CRM или таблица заявок.

## Размещение

Проект совместим с GitHub Pages, Netlify, Vercel и любым статическим хостингом.
